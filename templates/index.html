Building a PC

There are many aspects to think about when building a PC. While the task itself may seem a little daunting, building a PC can be a fun challenge to overcome and can help you understand quite a bit more about the machine you are using. Before getting started, there are a few things you may want to consider.

Budget

What is the overall budget for this machine? If you only need to build a PC for general office use, you won't need to spend as much money on parts. If you want to be able to do things like video/photo editing, gaming, etc. you may need to spend more money on higher performance parts, peripherals, and accessories.

Parts

There are quite a few parts that go into building a PC. Depending on what you intend to use the machine for, the kind of parts that you get will vary.The essential parts of a PC, however, are as follows:

  • CPU
  • Motherboard
  • GPU
  • Memory
  • Storage
  • Power Supply
  • Operating System(OS)
  • Cooling
  • PC Case

Tools

PC building isn't as complicated as working on your car. But just like any task that requires putting things together and taking them apart, you will need the right tools for the job.For PC building, the most useful thing you will need is a screwdriver. You will need to get in some hard to reach places, however, so make sure that you invest in a magnetic tipped screwdriver with a long shaft. A standard Phillips head #2 will work with all screws in a PC. It's very rare that you will be dealing with any screws bigger or smaller than the standard size.

These other tools aren't necessary, but you may find useful to have:

  • Scissors or a knife for opening packaging
  • Cable ties for cable management
  • Magnetic dish for keeping track of screws
  • Anti-Static Wristband
